# AI Emotion Detector

Real-time human emotion detection system using deep 
learning that identifies emotions through webcam feed.

## Features
- Real-time emotion detection
- 7 emotions: Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral
- Live webcam feed processing

## Tech Stack
- Python
- Keras
- TensorFlow
- OpenCV

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run: python emotion_detector.py
